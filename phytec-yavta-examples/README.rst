Yavta Examples
==============

Yavta stands for Yet Another Video Testing Application and can be used to
capture frames from a video device.

Sources can be found here:
https://git.ideasonboard.org/yavta.git/

Besides that v4l2-ctl can also be used to capture frames from a video device.


Scripts
-------
* yavta-capture-raw-full-csi1.sh
  Capture and save an 8 bit, raw, full resolution image from the connected
  camera on CSI1.

* yavta-capture-raw-full-csi2.sh
  Capture and save an 8 bit, raw, full resolution image from the connected
  camera on CSI2.

* yavta-capture-raw-select-csi1.sh
  Capture and save an image from the connected camera on CSI1 with
  the user being able to select the bit width and resolution from a
  predefined set of modes.

* yavta-capture-raw-select-csi2.sh
  Capture and save an image from the connected camera on CSI2 with
  the user being able to select the bit width and resolution from a
  predefined set of modes.
